# mainweb-aiyouxi-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容互不相关，每个页面均可独立访问和展示。仓库本身不针对任何特定网站或域名，仅作为通用页面存储与托管使用。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── page1/              # 示例页面目录
│   └── index.html
├── page2/
│   └── index.html
└── README.md           # 本文件
```

每个页面通常存放在独立的子目录中，便于管理和引用资源文件（如 CSS、JavaScript、图片等）。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独使用或删除。
- 归档的页面可能包含演示、工具、信息展示等类型的内容，具体用途请参考各页面内部说明。

## 维护说明

- 欢迎通过 Pull Request 提交新的独立 HTML 页面，或对现有页面进行修正。
- 新增页面请遵循以下基本规范：
  - 页面放在独立目录下，目录名应简洁明了。
  - 页面应包含基本的 HTML 结构（`<!DOCTYPE html>`、`<html>`、`<head>`、`<body>` 等）。
  - 避免在页面中引入外部不可信资源，确保页面可离线或低依赖运行。
- Issue 可用于报告页面功能问题或提出改进建议。
